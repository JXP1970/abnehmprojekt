# Projekt −10 kg

Eine eigenständige Single-File-Web-App zum Abnehmen im Bauplan-/Baustellen-Design.
Alles steckt in einer einzigen HTML-Datei — kein Build, kein Server, kein Konto.

## Funktionen

- **Planung** – Grunddaten erfassen, Grund-/Gesamtumsatz (BMR/TDEE), Tagesziel, Defizit und Zieltermin berechnen
- **Gewicht** – tägliche Messungen eintragen, Soll-Ist-Vergleich mit Verlaufsdiagramm und 7-Tage-Schnitt
- **Ernährung** – Tagesziele für Kalorien/Makros, Ampel-Liste „Was darf ich essen", zwei komplette 7-Tage-Essenspläne (Woche A/B umschaltbar), 20 aufklappbare Rezepte mit Zutaten und Zubereitung, Wochen-Einkaufsliste mit Mengen – alles skalierbar auf 1 oder 2 Personen – und optionales Tagebuch
- **Sport** – Wochenprogramm, zwei Kraft-Einheiten (Ganzkörper), Ausdauer, Trainings-Log
- **Woche** – wöchentlicher Gewohnheits-Check, Troubleshooting, Daten-Export (JSON) und Reset

## Nutzung

Die Datei [`index.html`](index.html) einfach im Browser öffnen –
oder online über GitHub Pages nutzen.

> **Speicherung:** Alle Eingaben werden im `localStorage` des jeweiligen Browsers gespeichert.
> Die Daten bleiben lokal auf deinem Gerät und werden **nicht** übertragen oder auf GitHub abgelegt.
> Ein anderer Browser oder Gerät hat einen eigenen, separaten Datenstand.
